# US_VISA_APPROVAL

## How to run

1. clone the repository

   ```cmd
   git clone https://github.com/Ashokyaganti27/US_VISA_APPROVAL.git 
   ```

2. Navigato to the project directory

    ```cmd
    cd US_VISA_APPROVAL
    ```

3. create and activate virtual environment

    ```cmd
    conda create -n us_visa python==3.8 -y
    ```

    ```cmd
    conda activate us_visa\
    ```

4. Install the required Dependendcies:

    ```cmd
    pip install requirements.txt
    ```
